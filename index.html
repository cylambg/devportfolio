Retrieving Weather Information

A demo project that exposes a REST API endpoint to retrieve weather information of a given city. When the endpoint was called, the program tried to retrieve weather information from two different websites, based on the availability of the city in the website. The response was cached and the cache was evicted every 3 seconds. The program was written in Java 11 with Spring Boot.

Sentiment Analysis on Trump’s Tweets

A coursework project of a sentiment analysis on Trump’s tweets to find out his opinions towards different topics and the popularity of his tweets in different emotions (positive, negative, or neutral). Data preprocessing and basic sentiment analysis using predefined tools was done before loading data into the models. Two models, K-Means Clustering and RNN Classification, were adopted for discussion of results and comparison of performance. Scripts were written in Python 3 with Jupyter Notebook.

View Project

Estimating Average Travel Time of Traffic Routes in Highways

A coursework project to estimate the average travel time from designated intersections to tollgates in highways in a 20-minute time window. Experiments and analysis were based on the given dataset of the average travel time of 6 different routes. Basic exploratory data analysis was done to understand the data. A pre-training model was built to estimate the average travel time of the missing records in the dataset. Two models, Feedforward Neural Network and XGBoost, were built to predict the average travel time and compare the results. Scripts were written in Python 3 with Jupyter Notebook.
